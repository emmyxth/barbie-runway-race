# Barbie Runway Race

A fun Barbie-themed game to plan your startup's headcount and runway. Drag employees onto the Barbie bus and see how long your runway lasts!

## How to run locally

Follow these steps:

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd barbie-runway-race

# Step 3: Install the necessary dependencies
npm install

# Step 4: Start the development server
npm run dev
```

## Technologies used

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Build for production

To build the project for production:

```sh
npm run build
```

The built files will be in the `dist` directory.
